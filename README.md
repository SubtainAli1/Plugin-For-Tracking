# Plugin-For-Tracking

Order Tracking Plugin – Quick Setup Guide

Follow the steps below to install, configure, and use the Order Tracking feature.

1. Install the Plugin

Go to WordPress Dashboard → Plugins → Add New

Click Upload Plugin

Select the plugin .zip file

Click Install Now and then Activate

2. Create the Tracking Page

Go to Pages → Add New

Name the page: Track Your Order

In the page content area, paste the following shortcode:

**[order_tracking]
**

Click Publish

This page is where customers will track their orders.

3. Add Tracking Information to an Order

Go to WooCommerce → Orders

Create a new order or select an existing one

Open the order

In the Order Tracking section:

Enter the Tracking Number

Select the Order Status
(Ordered, Dispatched, In Transit, or Delivered)

Click Update to save the order

4. Tracking an Order (Customer View)

Copy the Order ID from the Order list

Visit the Track Your Order page

Enter the Order ID in the tracker

Click Track

You will now see:

Current Order Status

Tracking Number

Live Map Location
